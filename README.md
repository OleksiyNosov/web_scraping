# Requirements
- Python 3.6.5

# Setup
## Install

```sh
sudo easy_install pip

sudo pip install --upgrade virtualenv

virtualenv --system-site-packages -p python3 .

pip install selenium

pip install bs4

pip install numpy

pip install pandas

pip install lxml

pip install tabulate

pip install plotly

pip install matplotlib
```

## Start

```sh
source ./bin/activate
```

## Stop

```sh
source deactivate
```

# Check

```sh
python3 -V
```
